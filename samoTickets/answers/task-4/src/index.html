Requirements

Create a slider which has 50 discrete positions representing 50 shades of gray.
Leftmost shade is white (255), rightmost shade is black (0).
A slider has a pointer (pictured in blue in mock) showing current slider position.

  1. Clicking on some shade moves slider pointer to that shade, changing slider value.
  2. It should be possible to move slider pointer with left/right arrow keys.
  3. Text field at the right (with green borders in mock) shows current value of a slider.
  4. Typing in the text field and pressing Enter updates slider selection, moving slider pointer to the closest shade.

Write JS in "src/react/Demo.jsx" or, if you prefer plain JavaScript, in "src/vanilla/Demo.jsx".

Write CSS in "src/slider.css". Inline styles are allowed, too.

"npm start" - starts dev server and opens this page.
