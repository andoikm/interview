Exercise 2

Implement format function.

format should accept a format string as a first argument and substitute next arguments into format string instead of {} placeholders.


Example: 
format('Hello {0} {1}', 'Mr.', 'X') // => 'Hello Mr. X'
format('{1}_{0}', '{1}', '{0}')     // => '{0}_{1}'

See browser console for more info.
