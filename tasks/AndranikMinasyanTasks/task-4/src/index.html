Requirements

"src/data.js" contains an array with tasks data. Render a Kanban board where each task is represented by card. See wireframe to understand how everything should look like.

  1. Tasks with same stage number should be rendered in the same column.
  2. Columns are sorted by stage number, in ascending order. Leftmost column has the lowest stage number, rightmost column has the highest stage number.
  3. Each card has button to move card to the left column (stage)
  4. Each card has button to move card to the right column (stage)
  5. Typing in quick filter field should filter cards by name and description

Write JS in "src/react/Board.jsx" or, if you prefer plain JavaScript, in "src/vanilla/Board.jsx".

Write CSS (if needed) in "src/board.css".

"npm start" - starts dev server and opens this page.
