Requirements

Using browser APIs and CSS create a player for playing a 24 fps animation sprite.
Player should infinitely loop animation.
Pause/Resume button at the bottom of the player should pause/resume animation.

  1. Animation sprite contains 300 frames, placed in 12 rows and 26 columns.
  2. Inside sprite frames are organized from left to right, on rows going from top to bottom.
  3. Width of each frame is 640px, height is 360px.

Write JS in "src/react/Demo.jsx" or, if you prefer plain JavaScript, in "src/vanilla/Demo.jsx".

Write CSS in "src/demo.css". Inline styles are allowed, too.

"npm start" - starts dev server and opens this page.
